# Elite Shine Detailing - Project Documentation

## Project Overview
This project is an ultra-premium, static single-page website for **Elite Shine Detailing**, a high-end auto detailing atelier based in București. It serves as a prestigious digital calling card, showcasing scientific-grade automotive care through a private concierge interface.

The application follows a minimalist Single Page Application (SPA) pattern using hash-based routing to navigate between key sections (Home, About, Services, Contact) without page reloads.

## Tech Stack
- **Frontend:** HTML5, CSS3 (Ultra-Premium Custom Styling), Vanilla JavaScript.
- **Styling:** Luxury aesthetic using 'Playfair Display' (headings) and 'Montserrat' (body). Features include glassmorphism, subtle SVG noise textures, and clinical spacing.
- **Animations:** High-prestige scroll-reveal system using the Intersection Observer API.
- **Language:** Romanian (`ro`).

## Architecture & Structure
- `index.html`: Core file containing all structural HTML, advanced CSS logic, and SPA routing.
- `imgs/`: Visual assets directory (4K-grade imagery for hero, services, and portfolio).

### Key Features
- **Concierge Interface:** Replaces traditional forms with direct, high-prestige communication channels (WhatsApp Concierge, Private Email).
- **Technical Masterclass:** Detailed service descriptions focusing on molecular-level care (SiO2/SiC coatings, Stage 3 correction).
- **Exclusivity Narrative:** Content designed to position the brand as a "by appointment only" elite atelier.
- **Responsive Luxury:** Perfected visual experience across all screen sizes.

## Development Workflow
### Running
- Open `index.html` in a modern browser. A local development server is recommended for the best experience with Intersection Observer animations.

### Conventions
- **Prestige Spacing:** Maintain high padding (typically `10rem`) between sections to preserve the "luxury breathing room" aesthetic.
- **Typography:** Headlines must strictly use 'Playfair Display' with low weights (400-600) for elegance.
- **Asset Integrity:** Always use high-resolution, professional imagery to match the ultra-premium UI.
